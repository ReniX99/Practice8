Для запуска проекта введите следующие команды:
1. Клонировать указанный репозиторий
```{git}
git clone https://github.com/ReniX99/Practice8
```

2. Запустить приложение:
```{bash}
cd Practice8
npm install
npm run build
npm run preview
```

3. Приложение будет доступно по URL:
```{bash}
http://localhost:4444/
```
